# CIASIE_nodejs
NodeJS project

## Installation
1. Installation des dépendances
```bash
cd api/ 
npm install 
```
2. Création du fichier de configuration
```bash
cp .env.example .env
```
3. Remplir le fichier de configuration
```bash
nano .env
```
4. Lancement de docker
```bash
docker-compose up
```
